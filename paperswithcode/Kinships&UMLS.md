# description
>The Kinships data set (Denham, 1973) describes the 26 different kinship relations of the Alyawarra tribe in Australia, among 104 individuals. The unified medical language system (UMLS) data set (McCray, 2003) represents 135 medical concepts and diseases, linked by 49 relations describing their interactions.
